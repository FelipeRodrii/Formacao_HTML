Breve Historia da Programação

Primeiros Computadores

Os primeiros Computadores que existiram, eram basicamente grandes máquinas de cálcular e que funcionavam a partir de Valvulas e que ocupavem basicamente uma sala toda. Com a melhoria tecnológica as valvulas foram em bvreve substituídas pelos transistos, isto diminuíu muito o espaço necessário dos computadores (tamanho de uma mesa mais ou menos). Depois com vinda dos capacitores e resistores esse tamanho diminuíu ainda mais, para o tamanho que vemos hoje.

Surgimento da Internet

A internet surgiu durante a guerra fria, com o objetivo de revolucionar a comunicação, se deu inicio dentro da DARPA (Defense Advanced Research Project Agency), neste inicio a comunicação era feita apenas por universidades e bases militares, utilizando comunicação por fio, essa rede foi chamada de ARPANET E apartir dai ela foi evoluindo até chegar o que é hoje.

Entendendo a Comunicação Client X Server

O que são Clients?

No linguajar de Web, Client é a parte que recebe a requisição feita, ou seja, normalmente o Client é o usuário que requisita dados ao Server.

Quando o Server devolve esses dados o nosso Navegador irá interpreta-los e nos entregar seu resultado. seja essses dados uma página Web, um arquivo de download Etc..

O que é Cache ?

Já que citamos Páginas Web, é interessantre ciatarmos uma das principais características delas em nosso sistema, que é o armazenamento em Cache.
Cache é a memória mais rápida do nosso sistema, porém ela tem uma baixa capacidade porém uma alte velocidade, no caso em questão, páginas Web guardam em nosso sistema informações de sua págima na memória Cache, como imagens por exemplo, isto faz com que na próxima vez que entremops página, ela levará menos tempo para carregar, claro, esses dados não temporários.

Aplicações Web

Aplicações Web, são fucionalidades que funcionam a partir da internet como meio de comunicação Client x Server que porém, não necessita de uma instalação local, como sites por exemplo (youtube, Facebook...), porém o mesmo não é dito de Apps, pois eles são instalados no celular. Se tratado da carreira profissional, são caminhos diferentes, porém, tem lá suas semelhanças
