Tags do HTML

O HTML é uma linguagem de marcação, feita para estruturmos textos e formulários, vamos apresentar nesta aba algumas das tags do HTML.

Tags de Titulos

Como o nome já diz, essas são a Tags para darmos titulos em nosso arquivo, essas tags vão de <h1> Titulo aqui</h1> até <h6> Titulo aqui</h6>, sendo uma hierarquia entre os titulos.
